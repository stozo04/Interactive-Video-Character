# Lessons Learned — 2ae9864e-f0d2-4532-a417-474a532c5ae7 — 2026-03-14

## Ticket
Fix admin dashboard redirect race caused by stale character loading in App.

## Codebase Discoveries
- `loadCharacters` in `src/App.tsx` unconditionally drives the top-level `view` state.

## Gotchas & Bugs
- React StrictMode double-invokes the mount effect, so the second async completion can override later navigation states.

## Approach That Worked
- Guarded the success path with a functional `setView` update that only changes state when still in `loading`.

## What Future Opey Should Know
- Any async state setter that can race user navigation should use a conditional functional update to avoid stale completions hijacking view state.
