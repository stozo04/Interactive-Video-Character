## Ticket
Trigger a server restart by touching the restart trigger file.

## Codebase Discoveries
- Server restart is triggered by writing to `server/.restart-trigger` so `tsx watch` detects the change.

## Gotchas & Bugs
- None encountered for this change.

## Approach That Worked
- Updated the restart trigger file with a new timestamp string to force a watcher restart.

## What Future Opey Should Know
- The restart endpoint simply touches `server/.restart-trigger`; direct file updates are sufficient.
